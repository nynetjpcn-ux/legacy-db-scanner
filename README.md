🔍 Legacy DB Scanner｜レガシーDBスキャナー
レガシー経理システムのデータを自動スキャンし、クラウド会計への移行可否レポートを生成するツールです。
「20年前の経理ソフトのデータ、本当に移行できるの？」に答えます。
できること
CSVファイルの全カラムを自動スキャン
SQLiteデータベース（.db）の全テーブルを解析
Excelファイル（.xlsx）のシートを一括チェック
NULL率・データ型・文字化けリスクを自動検出
Markdownレポートを自動生成
対応ファイル形式
形式
拡張子
備考
CSV
.csv
Shift-JIS / UTF-8 自動判定
SQLite
.db .sqlite
複数テーブル対応
Excel
.xlsx .xls
openpyxl が必要
使い方
# インストール（Excelも使う場合）
pip install openpyxl

# CSVファイルをスキャン
python legacy_db_scanner.py --file sales.csv

# フォルダ内の全ファイルをまとめてスキャン
python legacy_db_scanner.py --path ./data/

# レポートの出力先を指定
python legacy_db_scanner.py --file sales.csv --output my_report.md
